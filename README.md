# remit-cli
A CLI to perform basic remit actions

``` sh
npm install -g remit-cli
```

``` sh
$ remit-cli
  Commands:

    help [command...]                       Provides help for a given command.
    exit                                    Exits application.
    request [options] <endpoint> [args...]
    emit <endpoint> [args...]
    listen <endpoint>
```
